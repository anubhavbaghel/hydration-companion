# Hydration Companion App aka Hydra

Hydra is a smart mobile application designed to help users maintain healthy hydration habits throughout the day. The app provides reminders to drink water, and visualizes hydration progress to encourage consistent hydration.

Built with **React Native**, the app focuses on providing simple user experience.

---

## Features

* 🔔 **Smart Reminders** – Get timely notifications to stay hydrated
* 📊 **Progress Dashboard** – Visualize daily hydration goals
* ⚡ **Fast & Lightweight** – Built using modern React Native architecture
* 🧠 **Context-based State Management** for efficient data flow

---

## 🛠 Tech Stack

* **React Native**
* **Expo**
* **TypeScript**
* **React Context API**
* **Tailwind / NativeWind**
